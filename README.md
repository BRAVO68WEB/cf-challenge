# WorkerBin

![Logo](./website/public/favicon.ico)

A Pastebin alternative, but it is using CF Worker API to store the data.

# Code Stack Used

- Nodejs
- Nextjs
- TailwindCSS
- Wrangler

# CloudFlare Technologies Used

- Worker (as API)
- Worker KV (to Store data)
- Pages (for Deploying Frontend)
